## 📖 Project Overview

The **Productivity Tracker** is a full-featured Chrome Extension designed to help users improve focus and manage productivity efficiently.

The extension tracks time spent on websites, blocks distracting websites, and generates detailed productivity reports.

This project is developed using the **MERN Stack**:

* **MongoDB** – Database
* **Express.js** – Backend Framework
* **React.js** – Frontend UI
* **Node.js** – Server Runtime

Along with:

* Chrome Extension APIs
* REST APIs
* JWT Authentication

---

# ✨ Features

## ⏳ Website Time Tracking

* Tracks active tab usage in real-time
* Calculates time spent on each website
* Stores daily usage statistics
* Generates focus-time analytics

## 🚫 Website Blocking

* Add or remove distracting websites
* Automatically blocks selected domains
* Helps improve concentration and productivity
* Displays custom block messages

## 📊 Daily Productivity Reports

* Total focus time
* Most visited websites
* Most distracting websites
* Productivity percentage reports
* Visual productivity dashboard

## 👤 User Authentication

* Secure Login & Registration
* JWT Authentication
* User profile & preferences storage
* Cloud sync across devices

# ⚙ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/productivity-tracker.git
cd productivity-tracker
```

## 2️⃣ Install Dependencies

### Backend

```bash
cd server
npm install
```

### Frontend

```bash
cd client
npm install
```

---

# ▶ Run Application

## Start Backend Server

```bash
npm start
```

## Start Frontend

```bash
npm start
```

---

# 🧩 Load Chrome Extension

1. Open Chrome Browser
2. Go to:

```text
chrome://extensions/
```

3. Enable **Developer Mode**
4. Click **Load Unpacked**
5. Select the `extension` folder
